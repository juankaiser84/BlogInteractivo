CSS Variables y temas

2 Oct 2025 · por Juan Kaiser

Define tus paletas en :root y alterna con una clase (.theme--dark) en <html> o <body>. Guarda la preferencia en localStorage.

:root { --bg: #fff; --fg: #0f172a; }
.theme--dark { --bg: #0f172a; --fg: #e2e8f0; }
